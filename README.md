- [karabiner](https://karabiner-elements.pqrs.org/)
- [fish](https://fishshell.com/)
- [omf](https://github.com/oh-my-fish/oh-my-fish)
- [alacritty](https://github.com/alacritty/alacritty)
- [mononoki nerd font](https://github.com/ryanoasis/nerd-fonts/releases/download/v3.4.0/Mononoki.zip)
- tmux
- [tpm](https://github.com/tmux-plugins/tpm)
- neovim
- [zioxide](https://github.com/ajeetdsouza/zoxide)
- [asdf](https://asdf-vm.com/)
- [starship](https://starship.rs/)

### Installing
```bash
mv ~/.config ~/.config.backup
git clone https://github.com/laffed/.config.git ~/.config
```

Then selectively add what you require from `~/.config.backup`

Manually copy over any `~/.config/fish/fish_variables` from local source.
